## Name
Cardafy

## Description
Cardafy is a semi-blockchain marketplace with crypto wallet based login.

## Features
* CMS
* AI Chatbot

## Usage
```bash
git clone https://github.com/ArielJoe/Cardafy.git
npm install
```
Open sanity CMS in browser :
```bash
cd sanity
npm run dev
```
Open prisma studio in browser :
```bash
npx prisma studio
```
